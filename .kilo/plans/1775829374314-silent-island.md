# Plan: Add Validation to startDialog in ScoreSheet.vue

## Goal

Add validation in the `startDialog` method to check that all sections have been scored before the dialog is shown.

## Current Behavior

The `startDialog` method currently:

1. Calls `getScore()` to fetch the score data
2. Sets `showDialog = true` to open the dialog

## Required Change

Modify `startDialog` to:

1. After fetching the score, iterate through all sections where `max_score > 0`
2. Check that each section has a score value (> 0)
3. If any section is not scored, show a Quasar notification (`this.$q.notify`) with an error message
4. Only open the dialog if all sections have been scored

## Implementation Details

In `src/components/proposal/ScoreSheet.vue`, modify the `startDialog` method:

```javascript
startDialog() {
  this.getScore().then(() => {
    const unscoredSections = this.sections.filter(
      section => section.max_score > 0 && (!this.score[section.name] || this.score[section.name] === 0)
    );

    if (unscoredSections.length > 0) {
      this.$q.notify({
        type: 'negative',
        message: `Please score all sections before submitting. Missing: ${unscoredSections.map(s => s.title).join(', ')}`
      });
      return;
    }

    this.showDialog = true;
  });
}
```

Note: Need to ensure `getScore()` returns a promise or handle the async nature properly.
